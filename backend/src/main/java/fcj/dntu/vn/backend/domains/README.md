# domains

This folder contains application domains, for example:

```
domains
│   README.md
│
└───dtos
│   │   ApiResponseDto.java
│   │   BusDto.java
│   │   RouteDto.java
│   │
│   └───responses
│       │   LoginResponse.java
│       │   ...
│   
└───entities
│   │   BusEntity.java
│   │   ...
│   │
│   └───enums
│       │   RoleEnums.java
│       │   ...
```
